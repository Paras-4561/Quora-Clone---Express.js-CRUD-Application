# Express.js CRUD App (Quora Clone)

This is a simple **CRUD application** built with **Express.js**, **EJS**, and **UUID**.  
It works like a mini version of Quora, where users can:
- Create a new post
- View all posts
- Edit existing posts
- Delete posts
